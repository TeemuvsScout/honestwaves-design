HonestWaves Pixar v.2
PrimeVue + Tailwind

The dev team's Pixar.

A rebuild that uses PrimeVue components directly and Tailwind for the rest. No custom HTML where a PrimeVue control will do.

Pixar v.1 was the design specification. Pixar v.2 is the implementation reference — built with the exact stack your team is using: PrimeVue's component library + Tailwind for layout and spacing. The visual language is PrimeVue's Lara theme, with light brand customization (the Pixar violet accent + brand blue).

What changes from Pixar v.1

Same identity. Less custom code. Built on the components you already license.

PrimeVue components

Button, DataTable, Card, Tag, Dialog, Sidebar, TabView, Avatar, ToggleSwitch, AutoComplete, Toast — all PrimeVue. We stopped building div.btn when <Button> works.

Tailwind for layout

Grid, flex, spacing, typography — all Tailwind utilities. We use them around PrimeVue components, not inside them. grid grid-cols-4 gap-3 instead of a custom .kpis class.

Light brand customization

Only the brand blue and Pixar violet are pinned via CSS variables on :root. Spacing, radii, typography — we accept PrimeVue's defaults.
