HonestWaves dashboard redesign

Full UX + UI redesign of the HonestWaves dashboard for the operations staff using it every day. Modern enterprise aesthetic (Linear / Stripe / Vercel), compact density, ops-first information hierarchy. Buildable in the existing PrimeVue + Tailwind stack. Includes Travis @ Corewell's recommendations and the dev team's event catalog.
