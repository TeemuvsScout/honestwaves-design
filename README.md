# HonestWaves Dashboard Design

Prototypes and architecture diagrams for the HonestWaves smart locker dashboard, including the proposed Asset Accounting section.

## Live demo

The Master Hub is at https://teemuvsscout.github.io/honestwaves-design/HonestWaves_Master_Hub.html

## Files

- `HonestWaves_Master_Hub.html` — start here. Tabbed switcher with all prototypes and architecture diagrams.
- `HonestWaves_Asset_Accounting_Full.html` — 5 pages, 20 sub-views, 16 modals.
- `HonestWaves_Asset_Accounting_Design.html` — inline placements (Device Model, User profile, Kiosk header, return flow).
- `HonestWaves_Drill_Down_Pattern.html` — macro → micro navigation pattern.
- `HonestWaves_Tablet_Admin_Redesign.html` — on-locker operator pane redesign.
