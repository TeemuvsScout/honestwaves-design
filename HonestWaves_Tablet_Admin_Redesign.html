What was redesigned vs. today

Every change in this prototype maps back to a finding in the review doc:

  • Group → Departments (nav label matches the page title).
  • Unified 10-state palette. Every state has a distinct color and glyph (accessibility for color-blind users).
  • Sentence case for status pills ("Active", not "ACTIVE"). Friendly event names ("Bay reset", not "bay_reset" or "BAY_RESET").
  • Real human names ("Mark Rivera"), real time formats ("18h 49m"). No "user #12" or "1129m".
  • No "Exit App" in the primary nav. No floating back-arrow FAB. Kiosk SN and dev-channel hidden from default view (revealable via "Kiosk info" link in the footer).
  • Configuration: Locker mode, Late detection, Timeouts, Screensaver, and Designated mode flows are separate cards. A sticky save bar appears when there are unsaved changes.
  • Hardware test: explicit confirmation for destructive actions. UVC requires press-and-hold to start, with safety messaging.
  • System health: rebuilt as an operator dashboard with plain-English KPIs and sparklines. Raw telemetry (PID, MMKV, UART, queryAllBays) lives under a "Show engineering view" toggle.
  • Features: "(placeholder)" rows removed. "/dev/ttyS3", "APK", "HID" stripped. Test-scan affordances added.
  • Bays: filter chips for "With issues / Empty / In use", a "Bays needing attention" surface up top, and the live grid uses color + glyph (not color alone).
