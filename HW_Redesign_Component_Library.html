Pattern · 05

Sub-tabs (page-level)

Bound by data-pane="X". The matching pane ID gets .active on click.

Activity log 109
Device status
Scheduled 3
Custom report
Active pane: activity
Copy
<div class="subtabs">
  <div class="subtab active" data-pane="activity">
    Activity log <span class="count">109</span>
  </div>
  <div class="subtab" data-pane="status">
    Device status
  </div>
  <!-- ... -->
</div>

// JS pattern:
const tabs = document.querySelectorAll('.subtab[data-pane]');
tabs.forEach(t => t.addEventListener('click', () => {
  tabs.forEach(x => x.classList.toggle(
    'active', x === t
  ));
  // + show matching pane
}));
Sync pattern: a single query ([data-pane]) matches both top tabs and sidebar accordion items. One showPane() call updates both surfaces.
Pattern · 06

Sub-nav (sidebar accordion)

Multi-level. L2 chevrons toggle .expanded on the parent group. L3 supports its own chevron + nested L4 list.

Lockers
Lockers 12
Pink8 8 bays
Green1 8 bays
Blue3 32 bays
Locations 6
Copy
<a class="nav-item active">Lockers</a>
<div class="sub-nav">
  <div class="sub-nav-group expanded">
    <div class="sub-nav-item active" data-pane="lockers">
      <svg class="chev"
           onclick="event.stopPropagation();
                    this.closest('.sub-nav-group')
                        .classList.toggle('expanded');">…</svg>
      Lockers <span class="sub-count">12</span>
    </div>
    <div class="sub-nav-l3">
      <div class="sub-nav-item-l3">Pink8</div>
      <div class="sub-nav-item-l3">Green1</div>
    </div>
  </div>
</div>
Click the chevron above to expand/collapse the Lockers group. The chevron uses event.stopPropagation() so it doesn't trigger the parent item's navigation action.
